Тестовое задание для Avior Media

Реализована только главная страница. Целью было сделать адаптив под все устройства, имея только макет для экранов FullHD 1920px

## Для запуска проекта локально, нужно:

1. Иметь установленный на устройстве node.js от версии 22.18.0
2. npm i
3. npm run dev - для режима разработки / npm run start - для деплоя

Потыкать сайт можно на: https://avior-media.vercel.app/
